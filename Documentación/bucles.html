Tipos de bucles

Los bucles permiten realizar una tarea repetidamente siempre y cuando se cumpla una condición. Para trabajar con bucles se necesitan listas (arrays) u objetos, con contenido o vacíos, en función de la acción que queramos realizar. En JavaScript existen, principalmente, 2 tipos de bucles: for y while.

Bucles for

Se repite la tarea hasta que la condición es determinada como falsa o hasta que no haya más elementos en la lista u objeto sobre el que se está iterando. Hay diferentes formas de escribir un bucle for, y se utiliza una u otra en función del tipo de datos con el que estemos trabajando. Existen los bucles, for, for in, for of y forEach.

For

Esta es la forma básica de hacer un bucle. Se utiliza cuando tenemos una lista con un número fijo de elementos y sabemos cuántas veces queremos iterar sobre los elementos de la lista. Vamos a suponer que tenemos una lista vacía y queremos llenarla con números del 1 al 5. 

let num = 0;
for (var i = 0; i < 5; i++) {
	num++;
	console.log(num);
}

Hemos creado una variable 'num' con el número 0 y luego un bucle for. Dentro del bucle for le hemos dicho que cree una variable i igual a 0 var i = 0, es decir, el bucle empieza desde el 0. Luego le pasamos la condición, que en este caso es que i sea menor que 5 i < 5. Finalmente, le decimos que queremos que haga con el número, que en este caso es sumarle 1 i++. El resultado es que empieza desde el 0 y, siempre y cuando i sea menor que 5 (condición), le sumará uno a ese número y lo imprimirá en la consola.

For in

En este caso tenemos una lista u objeto con diferentes elementos y queremos iterar sobre cada elemento de la lista u objeto. El bucle terminará cuando no queden más elementos en la lista sobre los que iterar. Vamos a crear una lista con nombres y vamos a imprimir casa nombre individualmente. 

let nombres = ['Juan', 'Pedro', 'Paco', 'María']
for (nombre in nombres) {
	console.log(nombres[nombre])
}

La forma en la que se construye el bucle for in es mediante la palabra clave for seguida del nombre de la variable sobre la que iteramos, en singular ('nombre'), luego in y, finalmente, el nombre de la variable ('nombres'). Finalmente, para que imprima cada nombre de la lista tenemos que escribir console.log(nombres[nombre]). Como se puede observar, en lugar de devolver los nombres, devuelve el índice.

Este tipo de bucle también sirve para iterar sobre objetos. La sintaxis es la misma que para listas, la única diferencia es que, a la hora de obtener los resultado, si queremos ver la clave y el valor hay que escribir console.log(clave, persona[clave]). 

let persona = {nombre: 'Pedro', edad: 30}
for (clave in persona) {
	console.log(clave, persona[clave])
}

De esta forma podemos imprimir la clave y el valor de un objeto.

For of

El bucle for of es parecido al for in, la diferencia es que en for in, para obtener los valores de la lista hay que especificar nombres[nombre], mientras que el for of siempre te va a devolver los elementos de la lista directamente. Aún así, la forma de escribir el bucle es igual a la del bucle for in, a diferencia de que en lugar de in hay que escribir of. 

let nombres = ['Juan', 'Pedro', 'Paco', 'María']
for (nombre of nombres) {
	console.log(nombre)
}

ForEach

Este bucle es similar a los dos anteriores. Sirve para aplicar una función a cada elemento de la lista o simplemente obtener los datos de cada elemento de la lista. La forma en la que se escribe es la siguiente: 

let nombres = ['Juan', 'Pedro', 'Paco', 'María']
nombres.forEach((nombre) => {
	console.log(nombre.toUpperCase())
})

Escribimos el nombre de la variable que contenga la lista, seguido de .forEach(). En realidad .forEach es una función, por lo que tenemos que pasarle un argumento, que suele ser el nombre de la variable en singular, y luego la función que queremos realizar con los elementos de la lista, en este caso escribirlos en mayúsculas.

Bucles while

Los bucles while se repiten siempre y cuando no se le diga que pare, es decir, si no hay una condición en la que deben parar, son infinitos. En JavaScript hay dos tipos de bucles while: while y do while.

While

Los bucles while se ejecutan mientras una condición sea verdadera. Por ejemplo, vamos a crear un bucle que devuelva los números del 1 al 5. 

let num = 0;
while (num < 5) {
	num++;
	console.log(num)
}

En realidad la funcionalidad es la misma que la del bucle for, sin embargo, estos bucles necesitan un valor centinela que les diga cuándo deben parar. En el ejemplo superior el valor centinela sería el 5.

Bucle do while

El otro tipo de bucle while es el do while. Es similar al bucle while normal, pero la diferencia radica en que el do while garantiza que el bucle se ejecute al menos una vez, aunque el resultado sea indefinido. Vamos a ver un ejemplo: 

let num = 0;
	do {
	  num++;
	  console.log(num)
	} while (num < 5)

Como se puede observar el resultado es el mismo que utilizando el bucle while. Sin embargo, hay algunas diferencias en cuanto a la sintaxis y a la funcionalidad. En primer lugar, la forma en la que se escribe es mediante la palabra clave do, seguida de la función que queramos realizar, que en este caso es sumar 1 al número e imprimirlo en la consola. Finalmente, le tenemos que pasar la condición que se debe de cumplir para que se ejecute el bucle. Esto último lo haremos mediante la palabra clave while, seguida de la condición, que en este caso es que el número sea menor que 5. En segundo lugar, si cambiamos el contenido de la variable num por una cadena y ejecutamos el bucle, sabemos que no tiene sentido porque el bucle está pensado para variables numéricas. De hecho, un bucle while normal no daría ningún resultado, pero tampoco errores. Sin embargo, si utilizamos el bucle do while, como garantiza que el bucle se ejecute al menos una vez, el resultado que obtendremos es NaN o undefined.
