Deconstrucción de variables

La deconstrucción o desestructuración es una forma de extraer valores de listas o de objetos. Es muy útil cuando tenemos una lista con muchos valores y queremos deshacer la lista para poder utilizar los valores independientemente. Los valores extraídos se pueden asignar a variables.

Deconstrucción de listas

Para desestructurar una lista hay que declarar una variable cuyo nombre sean los nombres de las futuras variables independientes entre corchetes. Y esta variable tiene que ser igual al nombre de la variable que contiene la lista. Pongamos que tenemos una lista con colores y queremos desestructurar esa lista de forma que cada color quede asignado a una fruta. 

const colores = ['verde', 'amarillo', 'rojo']
const [manzana, plátano, fresa] = colores
console.log(manzana)
console.log(plátano)
console.log(fresa)

En este caso la lista original se llama 'colores', entonces tenemos que crear una variable nueva que contiene los nombres de las varaibles individuales que queremos crear ('manzana, plátano, fresa') y esto igualarlo a 'colores'. Como se puede observar se han creado 3 nuevas variables 'manzana', 'plátano' y 'fresa' que contienen los colores de la lista. Estas nuevas variables son de tipo cadena.

Deconstrucción de objetos

Lo mismo que hemos realizado con la lista se puede realizar con un objeto. 

const persona = {nombre: 'Paco', apellido: 'García', edad: 30, ciudad: 'Madrid'}
const {nombre, ciudad} = persona
console.log(nombre)
console.log(ciudad)

En este caso tenemos un objeto con información sobre una persona y queremos individualizar los datos. Si queremos extraer el nombre y la ciudad de la persona esos son los nombres que tenemos que pasar a la nueva variable que estamos creando. Es decir, si en lugar de {nombre, ciudad} ponemos otros valores que no se encuentran en el objeto, el resultado será undefined. En resto funciona igual que con los arrays, esta nueva variable debe de ser igual al nombre de la variable que contiene el objeto ('persona'). El resultado es que ahora 'nombre' y 'ciudad' son variables de tipo cadena.

La deconstrucción de objetos permite pasar elementos del objeto como argumentos de una función. Siguiendo con el ejemplo de los datos de la persona, podríamos pasar el nombre y el apellido de la persona como argumentos y generar una función que devuelva un saludo. Para pasar elementos de un objeto como argumentos debemos colocarlos entre corchetes. En realidad estos elementos pasarán a ser variables individuales de tipo cadena, en este caso. 

const persona = {nombre: 'Paco', apellido: 'García', edad: 30, ciudad: 'Madrid'}
const saludar = (({nombre, apellido}) => {
	console.log(`Hola ${nombre} ${apellido}`)})
saludar(persona)

Pasamos ({nombre, apellido}) como argumentos y, al llamar a la función pasamos la variable que contiene el objeto, en este caso 'persona'.

Intercambio de variables mediante deconstrucción

Por último, la deconstrucción también se puede utilizar para intercambiar el contenido de dos variables de forma sencilla. 

let manzana = 'amarillo';
let plátano = 'verde';
[manzana, plátano] = [plátano, manzana];

console.log(`
Manzana: ${manzana}
Plátano: ${plátano}
`);

Creamos dos variables, manzana y plátano, e igualamos dos listas con los nombres de esad variables, es decir, [manzana, plátano] = [plátano, manzana]. Como se puede observar, al imprimir los nombres de las variables, el contenido de ellas se ha intercambiado.

Un detalle importante es que al crear las variables debemos utilizar var o let. Si utilizamos const nos dará un error debido a la característica inmutable de las variables declaradas con const.
