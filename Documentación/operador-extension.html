Operador de extensión

El operador de extensión tiene múltiples usos que vamos a ir viendo. La forma en la que se expresa el operador es mediante tres puntos ... seguidos de una palabra.

Combinación de listas

El primer caso de uso que vamos a ver es la combinación de listas. El uso del operador de extensión permite juntar literalmente dos listas. Si utilizaramos el método tradicional para juntar dos listas, el resultado sería el siguiente: 

let numeros = [1, 2, 3, 4, 5]
let masNumeros = [6, 7, 8]
numeros.push(masNumeros)
console.log(numeros)

Como se puede observar, en realidad no combina ambas listas sino que mete la lista 'masNumeros' dentro de 'numeros'. Para obtener el resultado deseado, que es combinar ambas listas, debemos utilizar el operador de extensión. 

let numeros = [1, 2, 3, 4, 5]
let masNumeros = [6, 7, 8]
numeros.push(...masNumeros)
console.log(numeros)

Al añadir los tres puntos antes de la variable que queremos juntar (numeros.push(...masNumeros)), se combinan ambas listas, obteniendo así una única lista con los números del 1 al 8.

Copiar listas

El segundo uso del operador de extensión es copiar listas. Si en lugar de modificar la lista original, queremos crear una copia de esa lista original que además contenga la segunda lista, necesitaremos el operador. Para ello vamos a crear una nueva variable que contendrá la lista 'numeros'. Luego a esa lista añadiremos algunos números más. 

let numeros = [1, 2, 3, 4, 5]
let masNumeros = [6, 7, 8]
	
const copiaNumeros = [...numeros]
copiaNumeros.push(...masNumeros)
	
console.log(copiaNumeros)

En este caso lo que estoy haciendo es utilizar el operador para copiar la variable 'numero' en 'copiaNumeros' y, posteriormente, estoy utilizando el operador para combinar las dos listas. Ahora la lista de números del 1 al 8 ya no está en 'numeros' sino que está en una nueva variable que es 'copiaNumeros'.

Si en lugar de utilizar el operador de expansión le decimos const copiaNumeros = numeros, estaríamos sobreescribiendo la variable 'numeros'.

Como se puede observar, el contenido de la variable 'numeros' se ha sobreescrito, de forma que ahora 'numeros' y 'copiaNumeros' tienen el mismo contenido, la lista del 1 al 8.

Pasar argumentos a funciones

Otro uso del operador de extensión es para pasar argumentos a funciones. Vamos a probarlo con la función Math.max() que encuentra en número más grande entre una lista de números. 

console.log(Math.max(1, 3, 10, 8, 4))

Si pasamos los numeros directamente en la función, nos devuelve el resultado correcto, en este caso 10. Sin embargo, veamos lo que pasa si, en lugar de pasar los números directamente a la función, intentamos pasar una lista con los números. 

let numeros = [1, 3, 10, 8, 4]
console.log(Math.max(numeros))

Como se puede observar el resultado es NaN porque no entiende que lo que estamos pasando como argumento es una lista con números. Para poder pasar una lista con números y que el resultado sea el correcto, hay que utilizar el operador de extensión. 

let numeros = [1, 3, 10, 8, 4]
console.log(Math.max(...numeros))

Ahora entiende la lista como una serie de números y la función Math.max() devuelve el número más alto.

Deconstrucción de objetos

El último uso del operador es para deconstruir objetos. Vamos a suponer que tenemos un objeto con muchos elementos y queremos separarlo en variables. Queremos que los dos primeros elementos tengan cada uno una variable y, el resto de elementos, queden almacenados todos ellos en una tercera variable. 

const persona = {nombre: 'Paco', edad: 31, apellido: 'Pérez', ciudad: 'Madrid', coche: true}
const {nombre, edad, ...otrosDatos} = persona
console.log(nombre)
console.log(edad)
console.log(otrosDatos)

En primer lugar, creamos una variable objeto con diferentes características de una persona. Y luego, queremos separar los datos de forma que el nombre y la edad sean cada uno una variable, y el resto de datos queden en forma de lista en una tercera variable que se llame 'otrosDatos'. Para ello se puede utilizar la sintaxis que aparece en el ejemplo. Es decir, creamos una variable nombre, una variable y una variable otrosDatos y, a esta última le añadimos tres puntos antes de la palabra que indican que contendrá el resto de elementos de la lista original. Así, el resultado es una variable nombre que contiene 'Paco', una variale edad que contiene '31' y una variable otrosDatos que contiene el apellido, la ciudad y si tiene coche.
